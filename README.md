# A simple todo list
### React + Redux + SASS
#### This was made as a test for a job position :smirk:

##### The stack:
  - Webpack 2
  - React
  - react-semantic-ui
  - SASS
  - Babel

##### The features:
  - Add, Remove and Done item list
  - Order by newest or older
  - Filter by All and Completed
  - LocalStorage persistance

##### Running the project!

  ```javascript
  npm install
  npm build
  npm start
  ```
Open your browser on **localhost:8080**
